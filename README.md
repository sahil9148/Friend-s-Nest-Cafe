## 📖 About

**Friend's Nest Cafe** is a premium vegetarian cafe located in the heart of Janak Puri, New Delhi. This is the official website — a cinematic, award-winning single-page application built to showcase the full menu, allow online ordering via WhatsApp, and provide an AI-powered assistant named **Sia** for customer support.

> *"Where every bite is a memory and every sip starts a conversation."*

## ✨ Features

### 🎨 Design & Animations
- **Cinematic dark theme** with gold & red accent colors
- **Custom animated cursor** that morphs on hover
- **Floating ambient glow orbs** in the hero section
- **Scroll-triggered reveal animations** on every section
- **Infinite marquee ticker** showing all menu categories
- **Glassmorphism cards** with backdrop blur effects
- **3D particle system** — 70 animated particles with food emojis floating across the screen
- **Parallax photo gallery** with dual infinite scrolling rows

### 🛒 Live Ordering System
- **Add to Cart** — add/remove items, quantity controls
- **WhatsApp Checkout** — sends full cart directly to cafe WhatsApp (9220552344)
- **Order Form** — fill name, phone, address → sends via WhatsApp instantly
- **Toast notifications** on every cart action

### 🤖 Sia — AI Chatbot
- Powered by **Claude AI (Anthropic)** with full menu context
- Knows every dish, price, delivery info
- Can recommend items based on preferences
- Quick-reply buttons for common questions
- Typing indicator & smooth animations
- Available 24/7

### 📸 Photo Gallery
- **11 real food photos** from the cafe — enhanced & color-graded
- **5-panel expandable hero strip** — hover any panel to expand
- **Dual infinite scroll** rows in opposite directions
- **Fullscreen lightbox** with swipe support on mobile
- Touch-swipe to navigate between photos

### 📱 Fully Mobile Responsive
- **3 breakpoints**: Mobile (≤768px), Small phones (≤380px), Tablet (769–1024px)
- Touch cursor hidden on mobile devices
- Menu tabs scroll horizontally with touch
- Photo gallery cards resize for small screens
- Sia chatbot expands to full screen width on mobile
- Swipe left/right on lightbox photos
